## Run Locally

Clone the Name

```bash
  git clone https://github.com/EdgarHdzHdz17/FurryFriend.git
```

Go to the project directory

```bash
  cd Name FurryFriend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx expo start
```
